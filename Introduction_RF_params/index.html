Object

The goal of this document is to give a practical introduction to those specific parameters which have been devised to address the needs of RF engineering (Radio-Frequency) :
  • Transmission Line
  • Impedance Matching
  • Reflexion Coefficient
  • Smith Chart
  • Standing Waves
  • S Parameters (Scattering Parameters)
These constructs are useful each time circuits elements are interconnected by links the propagation time of which is not negligible compared to the timescale of the application signals.
We could say as well each time the system physical dimensions are not negligible compared to the typical wavelength of signals.
This extends widely beyond the field of wireless communication for which it was invented.

A few examples :
